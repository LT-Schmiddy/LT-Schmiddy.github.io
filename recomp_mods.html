Zelda64Recomp Mods

Zelda64Recomp is a fan-made port of The Legend of Zelda: Majora's Mask for modern Windows and Linux computers. The project uses the N64Recomp toolset created by Wiseguy on GitHub. In addition to many built-in improvements, The port features a powerful modding framework, allowing for new features and improvements to be added to the game.

Below is a list of every mod I've made for Zelda64Recomp, along with any relevant links.

-> EZ Text Replacer

ETZR is a library of my own design provides a simple and unified solution for other mods to replace in-game text and dialog, as Zelda64Recompiled does not offer one out of the box.
